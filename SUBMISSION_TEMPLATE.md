# GitHub Submission Description

## Project
React Frontend Intern Assignment - User Dashboard

## Repository Summary
This project is a responsive User Dashboard built with React functional components and hooks. It fetches users from JSONPlaceholder, provides search by name, supports client-side user creation, and uses React Context for global user state. User details are displayed on a dedicated route using React Router.

## Implemented Requirements
- Dashboard user list fetched from API
- Card UI showing name, email, phone, and company
- Search/filter by name
- Client-side "Create New User" form
- Global state via React Context API
- User Details page with route parameter
- Full details view including address and geo-location
- Mobile-friendly responsive layout

## Tech Stack
- React
- React Router DOM
- Fetch API
- CSS (responsive)
- Vite

## How to Run
```bash
npm install
npm run dev
```
